# Libertà – Scalable Social Platform

## Overview

Libertà is a scalable social platform developed for a real client project, designed with modular architecture and privacy-by-design principles.

The system focuses on secure interactions, clean separation of concerns and incremental product evolution.

This repository represents the structured and production-ready baseline of the application.

---

## Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase (PostgreSQL + Auth)
- Modular front-end architecture

---

## Architecture Principles

- Modular organization by domain
- Separation between UI, services and data layer
- Environment isolation (.env)
- Scalable backend integration
- Incremental release strategy (SemVer)
